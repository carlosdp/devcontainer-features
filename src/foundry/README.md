
# Foundry (Ethereum)

A feature that installs the [Foundry](https://github.com/foundry-rs/foundry) toolkit for Ethereum smart contract development.

## Example Usage

```json
"features": {
    "ghcr.io/carlosdp/devcontainer-features/foundry:1": {
        "version": "latest"
    }
}
```
